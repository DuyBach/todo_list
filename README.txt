Installguide for Django(Windows)
1. Install Python
2. Setup PATH
	1. C:\Python34\;C:\Python34\Scripts;
3. Install Setuptools
	1. Download ez_setup.py
	2. & python ez_setup.py
4. Install PIP
	1. & easy_install pip
5. Install Django
	1. & pip install django
